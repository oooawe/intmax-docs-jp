# コードリポジトリ

INTMAX Network の公式 GitHub リポジトリ一覧です。以下は、INTMAX チームとコミュニティが保守するコアコンポーネント、ツール、暗号ライブラリのリストです。スマートコントラクトや SDK から API サーバー、ゼロ知識証明システムまで、あらゆるリソースが揃っています。

## コア INTMAX Network リポジトリ

- **[`intmax2`](https://github.com/InternetMaximalism/intmax2)** — INTMAX L2 ネットワークのコアロジックとインフラストラクチャ
- **[`intmax2-function`](https://github.com/InternetMaximalism/intmax2-function)** — INTMAX オペレーション（Claim、リワード等）を支えるサーバーレスバックエンド関数
- **[`intmax2-withdrawal-aggregator`](https://github.com/InternetMaximalism/intmax2-withdrawal-aggregator)** — Ethereum への Withdrawal リクエストのアグリゲーターサービス
- **[`intmax2-claim-aggregator`](https://github.com/InternetMaximalism/intmax2-claim-aggregator)** — ユーザーからの Claim リクエストを集約・検証するサービス
- **[`intmax2-explorer-api`](https://github.com/InternetMaximalism/intmax2-explorer-api)** — INTMAX Block Explorer がトランザクション・ブロックデータを取得するための API バックエンド

## スマートコントラクト

- **[`intmax2-contract`](https://github.com/InternetMaximalism/intmax2-contract)** — INTMAX zk-Rollup ブリッジとネットワーク管理のメイン L1 スマートコントラクト
- **[`intmax2-reward-contract`](https://github.com/InternetMaximalism/intmax2-reward-contract)** — L1 上の Privacy Mining リワード配布コントラクト

## Zero-Knowledge Proof（ZKP）ライブラリ

- **[`intmax2-zkp`](https://github.com/InternetMaximalism/intmax2-zkp)** — INTMAX ネットワークで使用されるカスタム ZK 回路とロジック
- **[`plonky2-u32`](https://github.com/InternetMaximalism/plonky2-u32)** — Plonky2 の 32 ビット演算対応拡張
- **[`plonky2_bn254`](https://github.com/InternetMaximalism/plonky2_bn254)** — Ethereum 互換の BN254 楕円曲線を使用した Plonky2 実装
- **[`plonky2_keccak`](https://github.com/InternetMaximalism/plonky2_keccak)** — ZK フレンドリーな Keccak（SHA-3）ハッシュ関数の実装
- **[`polygon-plonky2`](https://github.com/InternetMaximalism/polygon-plonky2)** — Polygon の Plonky2 ライブラリのフォーク（INTMAX 固有の修正を含む）

## SDK

- **[`intmax2-client-sdk`](https://github.com/InternetMaximalism/intmax2-client-sdk)** — INTMAX ネットワークと連携するための TypeScript SDK（dApp 向け）
- **[`intmax2-e2e`](https://github.com/InternetMaximalism/intmax2-e2e)** — TypeScript と Rust を使用した INTMAX ネットワークのエンドツーエンドテスト
- **[`intmax-walletsdk`](https://github.com/InternetMaximalism/intmax-walletsdk)** — WalletSDK プロトコルを使用して INTMAX と統合するウォレットプロバイダー向け SDK

## ドキュメント

- **[`intmax2-docs`](https://github.com/InternetMaximalism/intmax2-docs)** — INTMAX Network の公式ドキュメント（開発者ガイド・API リファレンスを含む）
