# Installation

To get started with the `INTMAX WalletSDK` , developers can install it using various package managers like npm, yarn, or pnpm:

{% tabs %}
  {% tab title="npm" %}
    ```sh npm install intmax-walletsdk ```
  {% endtab %}
  {% tab title="yarn" %}
    ```sh yarn add intmax-walletsdk ```
  {% endtab %}
  {% tab title="pnpm" %}
    ```sh pnpm add intmax-walletsdk ```
  {% endtab %}
{% endtabs %}
