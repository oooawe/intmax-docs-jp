# Send Transaction

新しいトランザクションを作成・署名し、ネットワークに送信するメソッドです。`data` フィールドにコードが含まれている場合はメッセージコールトランザクションまたはコントラクト作成として処理し、`from` で指定されたアカウントで署名します。

**注意**：`to` アドレスが指定されていない場合、コントラクト作成トランザクションとみなされ、`data` フィールドがコントラクトの初期化コードとして使用されます。`gasPrice` は `maxPriorityFeePerGas` および `maxFeePerGas` と同時に使用できません。

## リクエストパラメータ

### Params\[0\]

| パラメータ | 型 | 必須 | 説明 |
| ---------- | ------- | ----------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `from`     | String  | Yes                                       | トランザクションの送信元アドレス（20 Bytes） |
| `to`       | String  | Yes（コントラクト作成時はオプション） | トランザクションの送信先アドレス（20 Bytes） |
| `gas`      | String  | No（デフォルト: 90000）                  | トランザクション実行に提供するガスの整数値。未使用分は返却される |
| `gasPrice` | String  | No                                        | マイナーに支払うガス価格（wei 単位） |
| `value`    | String  | Yes                                       | このトランザクションで送金する値（wei 単位）の整数値 |
| `data`     | String  | No                                        | コントラクトのコンパイル済みコード、またはメソッドシグネチャとエンコード済みパラメータのハッシュ |
| `nonce`    | Integer | No                                        | nonce の整数値。同じ nonce を使用するペンディング中のトランザクションを上書きできる |

## レスポンスパラメータ

| パラメータ | 型 | 説明 |
| --------- | ------ | ------------------------------------------------------------------------------------------ |
| `DATA`    | String | トランザクションハッシュ。トランザクションがまだ利用できない場合はゼロハッシュ（32 Bytes） |

## リクエスト例

```typescript
ethereum.request({
  method: "eth_sendTransaction",
  params: [
    {
      from: "0x5A3897b0513FdBeEc7C469D9aF4fA6C0752aBea7",
      to: "0xDeaDbeefdEAdbeefdEadbAcFdeadbeefDEADbEEF",
      value: "0x0",
      data: "0x",
    },
  ],
});
```
