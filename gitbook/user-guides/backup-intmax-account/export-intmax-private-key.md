# INTMAX 秘密鍵のエクスポート

INTMAX では、接続しているアカウントの秘密鍵（Private Key）をエクスポートできます。

## 重要事項

- 秘密鍵は、INTMAX Web App に接続しているウォレットによって生成されます。
- このエクスポート機能は**オフラインバックアップのみを目的**としており、Web App の別のブラウザセッションにインポートするためのものではありません。
- 同じ Ethereum アドレスを複数のウォレットにインポートし、同時に INTMAX Web App に接続することは**推奨しません**。

### 安全な保管方法

- **オフライン環境での保管を強く推奨**
- 物理的なセキュリティ（金庫、貸金庫など）を利用

## 手順

1. 歯車（⚙️）ボタンをクリック

![Export INTMAX Private Key](assets/user-guides/export_intmax_private_key_10.webp)

2. 設定メニューから「Backup INTMAX Addresses」を選択

![Export INTMAX Private Key](assets/user-guides/export_intmax_private_key_20.webp)

3. 「Export Private Key」をクリック

![Export INTMAX Private Key](assets/user-guides/export_intmax_private_key_30.webp)

4. 接続中のウォレットアプリで署名リクエストに応答

![Export INTMAX Private Key](assets/user-guides/export_intmax_private_key_40.webp)

5. 秘密鍵が表示されます。**必ず安全に保管してください。**
