## コントリビューション

開発者、デザイナー、研究者、そしてプライバシーとスケーラビリティに情熱を持つすべての方からのコントリビューションを歓迎します。

### コントリビューションの手順

1. **Fork & clone** — GitHub 上でリポジトリをフォークし、クローン
2. **ブランチを作成** — 修正や新機能用のブランチを作成
3. **変更をコミット** — 明確なメッセージを付けてコミット
4. **Pull Request を作成** — PR テンプレートに従って作成

送信前に [Contribution Guidelines](https://github.com/InternetMaximalism/intmax2/blob/main/CONTRIBUTING.md) をご確認ください。

---

## 行動規範（要約版）

すべてのコントリビューターにとって友好的、安全、かつ歓迎される環境を育むことに取り組んでいます。参加することにより、[行動規範](https://github.com/InternetMaximalism/intmax2/blob/main/CODE_OF_CONDUCT.md)に従うことに同意したものとみなされます。
