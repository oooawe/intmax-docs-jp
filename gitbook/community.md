## コントリビューション

開発者、デザイナー、研究者、そしてプライバシーとスケーラビリティに情熱を持つすべての方からのコントリビューションを歓迎します。

### コントリビューションの手順

1. **Fork & clone** — GitHub 上でリポジトリをフォークし、クローン
2. **ブランチを作成** — 修正や新機能用のブランチを作成
3. **変更をコミット** — 明確なメッセージを付けてコミット
4. **Pull Request を作成** — PR テンプレートに従って作成

まずは [Contribution Guidelines](https://github.com/InternetMaximalism/intmax2/blob/main/CONTRIBUTING.md) をご確認ください。

## 参加規約（要約版）

フレンドリーかつ安全、すべてのコントリビューターが歓迎される環境を育むことに取り組んでいます。

コントリビューターは、[参加規約](https://github.com/InternetMaximalism/intmax2/blob/main/CODE_OF_CONDUCT.md) に同意したものとみなされます
