# はじめに

## What is INTMAX?

{% hint style="success" icon="dna" %}
INTMAXとは、ゼロ知識証明（ZKP）とステートレスアーキテクチャにより、 **プライバシー**と **スケーラビリティ**とを両立させる Ethereum のレイヤー2プロトコル
{% endhint %}

ブロックチェーン上のプライバシーを実現するアプローチとして多くのプロジェクトは 「 _データを暗号化して守る_」方法を採用しています。  
ネットワーク上にステート（状態）を保持しつつ、暗号技術でその中身を秘匿するアプローチです。

**INTMAX** はそれらとは異なり *ネットワーク上にステート（状態）を保持しない* という完全なステートレス設計で構築／運用されています。  
データの正当性はゼロ知識証明によって検証され、ユーザー自身がプライバシーを秘匿したまま自分のデータ（ = 資産）を管理できます。  

この設計がもたらすのは、**構造的なプライバシー**で、保護すべきデータがネットワーク上に存在しない以上、漏洩や検閲のリスクそのものが排除されると同時にスケーラビリティも向上するという合理的な帰結（必然性）です。

プライベート転送や プライバシーマイニング（Privacy Mining） など、ステートレス設計がもたらす体験をぜひ公式サイトにてご確認ください。

### 特徴

- **構造的プライバシー** : ネットワーク上のプライバシーデータを暗号化で隠すのではなく、そもそも保存しない／存在させない
- **高いスケーラビリティ** : ステートレス設計によりノード負荷を最小化し、処理効率の大幅向上
- **低コストな送受信** : 誰もが、より速く・より安く・より安全にデジタル資産を扱える
- **ユーザー主権の設計思想** : データの管理主体はプロトコル側ではなくあくまでユーザー側
